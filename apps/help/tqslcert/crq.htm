Generating a Certificate Request

A certificate request is generated when you choose New Certificate Request from the File menu.

To create the certificate request, you must supply information about:

  • The issuer you expect to issue the certificate.
  • The call sign this certificate will be used for.
  • The DXCC entity that QSOs signed with this certificate will credit.
  • Your name and postal address.
  • Your email address.

In addition, certificate requests may be signed with an exisiting certificate if one is available.

A certificate request is generated via a "wizard" that takes you through the process step-by-step, displaying pages of information fields for you to fill in.

The first step of the wizard asks you to select the certificate issuer to which you will send this certificate request. (There may be only one issuer listed.)
