Localizing TQSL

In TQSL 2.1, the text that appears in TQSL's windows, dialogs, and error messages can be presented in English, French, Italian, Japanese, German, Portuguese, or Spanish. Translations were accomplished using Google Translate, and thus would benefit from improvements by fluent speakers.

If you'd like to help improve one or more translations,
  1. if you're not already a member, join the ARRL-LoTW Yahoo Group by sending an email message to ARRL-LOTW-owner@yahoogroups.com
  2. post a message on the ARRL-LoTW Yahoo Group expressing your interest by sending an email message to ARRL-LOTW@yahoogroups.com
