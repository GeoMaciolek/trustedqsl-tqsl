LotW Files and Icons

LoTW uses four kinds of files, distinguished by Filename Extension.

  • on Windows, Windows Explorer displays these files with an Icon and Type:
    Filename ExtensionIconFile TypeCreated ByUsage
    .tq5 Callsign Certificate RequestTQSLCertCallsign Certificate Request file submitted to ARRL
    .tq6 Callsign CertificateARRLCallsign Certificate file referenced by TQSL and managed by TQSLCert (cannot be backed-up, recovered, or moved to another PC)
    .tq8 Digitally Signed LogTQSL or logging applicationDigitally Signed Log file submitted to LoTW
    .p12 Personal Information ExchangeTQSLCertCertificate Container managed by TQSLCert (also referred to as a PKCS#12 file; can be backed-up, recovered, or moved to another PC)
