Preferences

TQSL options and configuration settings are reached via the File menu's Preferences command.

The Preferences dialog contains several tabs, each of which provides particular settings:

  • Options - Settings for file types and processing options.

  • ADIF Modes - Mappings of ADIF log file MODE field values to TrustedQSL modes.

  • Cabrillo Specs - Settings that describe the contest-specific parts of Cabrillo log files.
