Preferences - ADIF Modes

The ADIF MODE field is defined by the ADIF specification to have only a particular set of mode values. These values are not exactly the same as the mode values supported by TrustedQSL, but TQSL knows about that and will map defined ADIF modes to the appropriate TrustedQSL modes automatically.

However, many logging programs do not restrict the contents of the MODE field to the values listed in the ADIF specification. To handle that problem, the ADIF Modes tab of the Preferences dialog lets you map a MODE field value to one of the valid TrustedQSL modes.

A list of mappings is displayed, showing the present mappings. For example:

In this example, when BPSK appears in the MODE field of an ADIF record during signing, PSK31 will be output in the TrustedQSL file. Similarly, a mode of MCW will be output as CW.

To delete a mapping, click on the list entry to select it, then choose Delete.

To add a mapping, choose Add to get the Add ADIF Mode Dialog:

Type the ADIF mode into the ADIF Mode field. Then select the desired TrustedQSL mode from the list and choose OK.
