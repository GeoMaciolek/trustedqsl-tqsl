Checking for Updates

To determine if a new version of TQSL is available, select the Help menu's Check for Updates... command. If an update is available, the Update Available dialog will appear, showing

  • the URL from which the new version can be downloaded
  • the URL at which a description of the new version can be reviewed

If no update is available, the No Updates dialog will appear, stating that the current version is the newest available.

Click the OK button to dismiss either dialog.
