Moving Callsign Certificates and Station Locations from One Computer to Another

If you have TQSL and TQSLCert running on one computer (the source) and wish to set it up on a second computer (the destination), you must

  • Install TQSL and TQSLCert on the destination computer
  • Copy each of your Callsign Certificates from the source to the destination
  • Copy each of your Station Locations from the source to the destination

Moving a Callsign Certificate to Another Computer
A Callsign Certificate file, whose filename extension is .tq6, cannot be simply copied from one computer to another. Only a Certificate Container file, whose extension is .p12, can be moved in this manner.

1.If you have not created a Certificate Container file for the Callsign Certificate you wish to move, do so by following the instructions in step 3.f
2.Copy the Certificate Container file from the source computer to the destination computer
3.On the destination computer, run the TQSLCert application
   3.aOpen the File menu and select the Load Certificate File command; the Load Certificate file dialog will appear
   3.bIn the Load Certificate file dialog, Select PKCS#12(.p12) certificate file, and click the Next button; the Select Certificate File dialog will appear.
   3.cIn the Select Certificate File dialog, select the Certificate Container file you copied in step 2 above, and click the Open button
   3.dIn the Load Certificate file dialog, enter the password you specified when you created the Certificate Container file, and click the Next button; the New Password dialog will appear.
   3.eIn the New Password dialog, specify a password in both textboxes (you can use the same password you supplied in step 3.d), and click the Ok button; the Load Certificate file dialog should display Loading complete
   3.bIn the Load Certificate file dialog, click the Finish button
   3.bIn the tQSL Certificates window, select the File menu's Exit command

Moving a Station Location to Another Computer
Windows

TQSL stores all of your Station Locations in a file named station_data, which can simply be copied from one computer to another. No matter what version of Windows you're using, you can navigate to the source computer's folder that contains this file and the destination computer's folder that will receive this file with the following two steps in Windows Explorer:

  1. double-click Computer or My Computer
  2. In Windows Explorer's Address bar, type %AppData%\TrustedQSL and strike the Enter key

Apple OS X

If the operating system is OS X, the station_data file resides in the folder ~/.tqsl

Linux

If the operating system is Linux, the station_data file resides in the folder ~/.tqsl

Copying the station_data file

After you've found the station_data file on source computer and identified the proper location on the destination computer, simply copy the file from the source to the proper folder on the destination. Then start TQSL on the destination computer, open the Station menu, and select the Edit locations command; the Edit Station Locations dialog should now list your station locations.
