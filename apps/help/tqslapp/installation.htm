Installing TQSL and TQSLCert

The TQSL and TQSLCert applications used by LoTW are free to download and use, and are available for Microsoft Windows and Apple OS X. If you are running anti-spyware or anti-malware applications such as Bit Defender, configure these applications to allow both TQSL and TQSLCert to run.

Installing TQSL and TQSLCert on Microsoft Windows
TQSL and TQSLCert will run on the following versions of Microsoft Windows:
  • Windows 2000
  • Windows NT
  • Windows XP (32-bit and 64-bit)
  • Windows Vista* (32-bit and 64-bit)
  • Windows 7* (32-bit and 64-bit)
  • Windows 8* (32-bit and 64-bit)
Download TQSL and TQSLCert for Windows here .

     * Users of Microsoft Windows Vista, Windows 7, and Windows 8 users should download the tqsl-114.msi installer and start it with Run as Administrator

Installing TQSL and TQSLCert on Apple OS X
TQSL and TQSLCert will run on both PowerPC and Intel processors on OS X version 10.4 and later. Download TQSL and TQSLCert for OS X here .

Building TQSL and TQSLCert for Linux
Linux users can build the software from the source code for the tqsllib library, and the TQSL and TQSLCert applications. ARRL does not maintain packages for the many Linux distributions. For assistance with obtaining a packaged version for your Linux distribution, contact your distribution's maintainer.
