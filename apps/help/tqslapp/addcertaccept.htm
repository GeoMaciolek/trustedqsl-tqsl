Accepting an Additional Callsign Certificate

These instructions assume that your Callsign Certificates reside in the folder C:\MyLoTWCertificates (if you've stored them in a different folder, adjust the instructions accordingly).
1.You will receive email from the ARRL containing an attached file named theCallsign.tq6 -- save this fileinto the folder C:\MyLoTWCertificates.
2.To proceed, the file theCallsign.tq5 must be present in C:\MyLoTWCertificates; if you deleted this file, you'll have to start over by requesting a new certificate from the ARRL.
3.Start TQSL and select the Callsign Certificates tab, which lists your Callsign Certificates.
   3.aopen the File menu and select the Load Callsign Certificate File command; the Load Certificate File window will appear
   3.bin the Load Certificate File window, set the type of certificate to TQSL (.tq6), and click the Next button; the Select Certificate File window will appear
   3.cin the Select Certificate File window, select the file C:\MyLoTWCertificates\theCallsign.tq6 and click the Open button
   3.din the Load Certificate File window, click the Finish button
   3.ethe tQSL Certificates window should now show an entry for your certificate with a yellow medal icon:
       3.e.1.click on this entry to select it
       3.e.2.open the Certificate menu and select the Save command; the Enter PKCS#12 file to save to window will appear
       3.e.3.in the Enter PKCS#12 file to save to window, set the Save in folder to C:\MyLoTWCertificates, set the File name to theCallsign.p12, and click the Save button; a PKCS#12 Password window will appear
       3.e.4.in the PKCS#12 Password window, specify a password in both textboxes, and click the OK button; be sure to remember this password, as you'll be prompted to present it when using this Callsign Certificate to submit QSOs to LotW.
   3.fyou can terminate tQSL by opening its File menau and selecting the Exit command
4.Configure your Backup and Restore application to include C:\MyLoTWCertificates\theCallsign.p12 in the list of files it backs up.
5.At the completion of these actions, the folder C:\MyLoTWCertificates will contain
   *the file theCallsign.tq6 -- contains the actual Callsign Certificate, but in a form that cannot be moved to another folder or saved on an archival storage device
   *the file theCallsign.p12 -- contains the Callsign Certificate in a form that can be moved, backed-up, and recovered; you can use the tQSL Certificates application to extract YourCallsign.tq6 from YourCallsign.tq12 if needed
