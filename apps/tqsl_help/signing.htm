Preparing TrustedQSL Files

Once you've defined a station location you are ready to sign QSO records for that location.

N.B. All of the records in the ADIF or Cabrillo file you wish to sign must be for QSOs from the same station location. If the file contains records from multiple locations, use your logging software (or text editor) to change the file or create a new one that contains records from only one station location.
