TQSL Utility

The TQSL utility program is used to prepare files for the TrustedQSL system.

Preparing a TrustedQSL file requires associating one of more QSO data records with station locations, then digitally signing the associated records and formatting the result into TrustedQSL records.

TQSL allows you to:

  • Add, delete and modify station locations, based on the TrustedQSL signing certificates you have available.

  • Enter and edit QSO data records as ADIF files.

  • Sign the data contained in ADIF or Cabrillo log files to produce TrustedQSL files.
